MIKU ESKPO

MIKU ESKPO was a final project for Human Computer Interaction course that I took during my 4th semester. This final project was done in a group of 4. This website is an online platform for buying concert tickets the name MIKU ESKPO was one of the requirements in the question paper of the final project. I was responsible for developing 2 pages in the website which was Home Page and About Us Page using HTML, CSS, and JavaScript and I used sublime text as the text editor. In addition, I designed for the logo using Adobe Photoshop 2020.
